2. Bases

2.1 Portée du traitement des données

En principe, nous ne traitons les données personnelles de nos utilisateurs que s'il est nécessaire de fournir un site Web fonctionnel ainsi que nos autres services. Le traitement des données personnelles n'a lieu qu'après accord de l'utilisateur. La seule exception à cette règle est lorsqu'un accord préalable est techniquement impossible et qu'un traitement est autorisé par la loi.

2.2 Base légale du traitement

Art. 6 (1) lit. un RPGD sert de base légale pour tout traitement pour lequel nous obtenons un consentement spécifique. Si le traitement de données à caractère personnel est nécessaire à l'exécution d'un contrat auquel la personne concernée est partie ou à fournir tout autre service, le traitement est basé sur l'article 6, paragraphe 1, de la loi. b GDPR. Il en va de même pour les traitements nécessaires à la réalisation de mesures précontractuelles, par exemple dans le cas de demandes concernant nos produits ou services. Notre société est-elle soumise à une obligation légale en vertu de laquelle le traitement des données à caractère personnel est requis, par exemple pour le respect des obligations fiscales, le traitement est basé sur l'Art. 6 (1) lit. c GDPR. Dans les rares cas où le traitement de données à caractère personnel est nécessaire pour protéger les intérêts vitaux de la personne concernée ou d'une autre personne physique, le traitement serait fondé sur l'art. 6 (1) lit. d GDPR. Enfin, si le traitement est nécessaire aux fins des intérêts légitimes de notre société ou d'un tiers, les opérations pourraient être fondées sur l'article 6 (1) lit. f GDPR, sauf dans les cas où ces intérêts sont outrepassés par les intérêts ou les libertés et droits fondamentaux de la personne concernée.

2.3 Période pendant laquelle les données personnelles seront stockées

Les critères utilisés pour déterminer la période de stockage des données à caractère personnel sont la période de conservation légale respective. À l'expiration de cette période, les données correspondantes sont systématiquement supprimées, dans la mesure où elles ne sont plus nécessaires à l'exécution du contrat ou à la conclusion d'un contrat.

3. Servir notre site Web ainsi que les mises à jour de l'application, la création de fichiers journaux

3.1 Portée du traitement des données

Lors de chaque demande adressée à notre site Web et lors de l'utilisation de l'application pour l'actualisation récente des données de configuration, notre système traite automatiquement les informations provenant du système informatique de votre appareil :

  • Le type de navigateur, le système d'exploitation et la version utilisés
  • Le fournisseur de services Internet du système accédant
  • Une adresse de Protocole Internet (adresse IP)
  • La date et l'heure d'accès au site Internet
  • Le site Web à partir duquel un système d'accès parvient à notre site Web (dénommés "référents")
  • Sous-sites

Certaines de ces données sont conservées pour enregistrer des fichiers sur notre système. Nous ne </ strong> conservons pas les adresses IP ou autres données permettant de tirer des conclusions sur la personne concernée. Les données anonymes des fichiers journaux du serveur sont stockées séparément de toutes les données personnelles fournies par une personne concernée.

3.2 Base légale du traitement

La base légale pour ce traitement est l'art. 6 (1) lit. f GDPR.

3.3 Finalité du traitement de données

Nous devons traiter ces données pour pouvoir vous servir le contenu du site Web ou de l'application. Nous devons enregistrer l'adresse IP pour l'heure de la session afin de vous envoyer le contenu du site Web. Par conséquent, nous pouvons traiter ces données conformément à l'art. 6 (1) lit. f GDPR.

3.4 Période pour laquelle les données personnelles seront stockées

Le responsable du traitement des données ne traite et ne stocke les données à caractère personnel de la personne concernée que pendant la période nécessaire à la conservation. En particulier, nous ne stockons pas ces données plus longtemps que le temps de votre session sur notre site Web.

3.5 Droit d'objection

Le traitement de ces données est nécessaire au fonctionnement du site Web. Par conséquent, vous ne pouvez pas vous désinscrire de ce traitement.

4 Analyses utilisant Matomo (ehemals PIWIK)

4.1 Portée du traitement des données

Sur ce site Web, le contrôleur a intégré le composant Matomo. Matomo est un outil logiciel open-source pour l'analyse Web. L'analyse Web est la collecte, la collecte et l'évaluation de données sur le comportement des visiteurs provenant de sites Internet. Un outil d’analyse de sites Web recueille, entre autres, des données sur le site Web à partir desquelles une personne concernée a accédé à un site Web (dénommé "référent"), à quelles pages du site Web ont été consultées ou à quelle fréquence et pendant quelle période une sous-page a été consultée. vu. Une analyse Web est principalement utilisée pour l'optimisation d'un site Web. En particulier, nous stockons :

  • Premiers octets de l'adresse IP de l'utilisateur (c'est-à-dire 123.234.0.0 au lieu de 123.234.215.78)
  • Le site Web demandé par l'utilisateur
  • Le site Web de l'utilisateur (appelé "référent")
  • Les sous-pages demandées
  • La durée du séjour de l'utilisateur sur le site
  • Le nombre de requêtes sur le site

Le logiciel est exploité sur le serveur du contrôleur, les fichiers journaux sensibles à la protection des données sont stockés exclusivement sur ce serveur. Le logiciel est configuré pour anonymiser les adresses IP en supprimant la dernière moitié d’octets. De cette façon, nous ne pouvons pas retracer les données vers un périphérique ou une connexion individuelle. Le composant Matomo a pour objet l'analyse des flux de visiteurs sur notre site Web. Le responsable du traitement utilise les données et informations obtenues, entre autres, pour évaluer l'utilisation de ce site Web afin de compiler des rapports en ligne présentant les activités sur nos pages Internet.

Matomo installe un cookie sur le système informatique de la personne concernée. Avec la configuration du cookie, une analyse de l'utilisation de notre site Web est activée. Lors de chaque appel à l'une des pages individuelles de ce site Web, le navigateur Internet du système informatique de la personne concernée par le biais du composant Matomo est automatiquement invité à soumettre des données à des fins d'analyse en ligne sur notre serveur. Au cours de cette procédure technique, nous obtenons des informations sur des informations personnelles, telles que l'adresse IP de la personne concernée, qui permettent de comprendre l'origine des visiteurs et des clics.

4.2 Base légale du traitement

La base légale pour ce traitement est l'art. 6 (1) lit. f GDPR.

4.3 Finalités du traitement de données

Le traitement de ces données nous permet d'analyser l'utilisation de notre site Web. Cela nous permet d'améliorer constamment la convivialité et l'utilité de notre site Web. Pour cette raison, nous avons un intérêt à traiter ces données et pouvons le faire en vertu de l'art. 6 (1) lit. f GDPR. En anonymisant les adresses IP, nous respectons l'intérêt de l'utilisateur de protéger ses données de manière raisonnable.

4.4 Période pendant laquelle les données personnelles seront stockées

Nous allons supprimer tous les jeux de données individuels après 90 jours. Passé ce délai, nous n'archiverons que les moyennes et les sommes journalières, mais rien ne pourra être analysé pour chaque visiteur.

4.5 Droit d'objection

Comme indiqué ci-dessus, la personne concernée peut à tout moment empêcher l’installation de cookies sur notre site Web par un ajustement correspondant du navigateur Web utilisé et ainsi refuser définitivement l’installation de cookies. Un tel ajustement du navigateur Internet utilisé empêcherait également Matomo de créer un cookie sur le système informatique de la personne concernée. En outre, les cookies déjà utilisés par Matomo peuvent être supprimés à tout moment via un navigateur Web ou un autre logiciel.

En outre, la personne concernée a la possibilité de s’opposer à une collection de données relatives à une utilisation de ce site Internet générées par Matomo, ainsi qu’au traitement de ces données par Matomo et à la possibilité de s’y opposer. Pour cela, la personne concernée doit définir un cookie de désinscription en cliquant sur un lien du paragraphe suivant. Le cookie de désinscription défini à cet effet est placé dans le système informatique utilisé par la personne concernée. Si les cookies sont supprimés sur le système de la personne concernée, celle-ci doit rappeler le lien et définir un nouveau cookie de désinscription.

5. Droits de la personne concernée

Si des données personnelles vous concernant sont traitées, vous êtes une personne concernée au sens du GDPR.

Étant donné que nous ne stockons aucune donnée personnelle que nous pourrons ultérieurement vous connecter en tant que personne, nous ne pouvons pas supprimer, rectifier, effacer, restreindre ou divulguer des données personnelles spécifiques.

5.1 Droit de se plaindre

Vous pouvez toujours déposer une plainte auprès des autorités de protection des données de l'État membre dans lequel vous résidez, travaillez ou que nous résidons, si vous pensez que notre utilisation des données à caractère personnel enfreint le RPGD.
